Inserindo Vídeos Localmente

Este vídeo está hospedado no meu próprio servidor.

Usei a tag <video> com o parâmetro width em 500 para deixar o vídeo em um tamanho adequado ao site. Usei o parâmetro poster para colocar uma capa inicial no vídeo e o parâmetro controls para deixar o controle do vídeo visual.

Também usei a tag <source> dentro da tag <video> para deixar mais de um vídeo disponível, para que seja executado nos principais navegadores. Porém esta opção pode deixar a hospedagem bem mais cara ao cliente.
