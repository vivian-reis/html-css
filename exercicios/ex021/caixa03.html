Teste de Borda

O que eu aprendi?

Como eu não consegui criar uma borda pelo Gimp, criei pelo site https://www.favicon.cc/.

Para usar a borda criada, segui o passo a passo abaixo:

  1. Usei o parâmetgro border, que é necessário para o sistema reconhecer a borda personalizada nos passos seguintes;
  2. Usei o parâmetro 'border-image-source' para localizar o arquivo com a borda desenhada;
  3. Usei o parâmetro 'border-image-slice'(inicialmente em 20px), para "fatiar" a borda criada. Este recurso faz com que a borda se espalhe pela caixa selecionada.
  4. Usei o parâmetro 'border-image-repeat em 'repeat' para igualar o desenho por toda a borda.
  5. Por sim, usei o DevTools para ajustar os parâmetros do CSS.
