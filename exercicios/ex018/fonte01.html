Título

Aprendi a exportar tipos de fontes do site https://fonts.google.com/. Pra isso, depois de selecionar a fonte desejada, exportei o @ com as regras de cada fonte para dentro do campo style no css. Também usei as regras para CSS que foram disponibilizadas no próprio site.

Para o título usei a fonte Fredericka the Great, no tamanho 3em, com "peso" normal. Para o subtítulo usei a fonte Kaushan Script, também com tamanho 3em, mas sem especificar o "peso".

Subtítulo

Já nos parágrafos usei a fonte Work Sans, com tamanho 16px.

Geralmente usam-se 3 tipos de fontes:

  1. Para grandes títulos;
  2. Para destaques;
  3. Para textos normais.

É importante não exagerar na quantidade de fontes selecionadas, no máximo 4 familias de fontes!
