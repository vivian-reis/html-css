Imagem Dinâmica

Tarefa: tente abrir este site em vários dispositivos diferentes ou simplesmente aumente e diminua o tamanho do seu navegador.

Para esta imagem dinâmica, dentro da tag <picture> usei a tag <source:media:type>, com o parâmetro media no máximo (max-width) e não no mínimo e com mais 50px acima do tamanho das imagens.

Também adicionei o caminho do arquivo no parâmetro srcset e o tipo 'imagem/png' em type.

Outra coisa importante é que precisa ser em ordem crescente para que todas as imagens carreguem adequadamente.

Reproduzindo Áudios

Vamos aprender a reproduzir áudios em HTML:5.

No áudio acima usei a tag <audio>, que é a maneira mais simples de se adicionar um arquivo de áudio em um site. Basta informar o endereço do link no parâmetro src e adicionar o parâmetro controls autoplay. Mas não temos garantia de que o áudio será reproduzido adequadamente em todos os navegadores.

Por isso, o recomendado é usar a tag abaixo:

Adicionamos tags <source>entre as tags <audio>, preenchemos o parâmetro src com o endereço do arquivo de mídia e o campo type com o tipo de mídia: audio/mpeg para MP3 e audio/ogg para OGG. O arquivo do tipo wav não é recomendado porque são grandes e pesados. Este modelo pode ser usado como download.

Além disso, dentro da tag <audio>, incluímos o parâmetro preload com metadata porque só carregará algumas informações básicas do arquivo. E a opção auto (ou nenhuma) significa que o site só executará o áudio depois de baixá-lo por completo (o que é muito ruim para arquivos grandes). E a opção none não carrega nada.

O loop irá reproduzir o áudio automaticamente.
