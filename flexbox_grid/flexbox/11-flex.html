A ordem da shorthand é: grow - shrink - basis

E o padrão é: 0 (grow) 1 (shrink) auto (basis).

flex: n 1 auto;

Os 2 primeiros itens tem flex 1 1 auto.

O terceiro item tem flex 2 1 auto.

O quarto item tem flex 1 1 auto e width de 60px.

Já o último item tem flex 0 1 auto, mas por ser o padrão, não faz diferença esta configuração.

flex: 1;

Flex = 1 (grow 1, shrink 1, basis 0)

Por isso, os ítens ficam com a mesma altura, devido a quebra de linha do primeiro item.

Os dois primeiros itens estão com basis zero.

O terceiro item está com flex 2 (grow 2, shrink 1, basis zero).

O quarto item possui flex 1, width 720px e min-width de 80px. Como o basis é zero, o width é ignorado, mas o min-width é mantido.

O quinto item possui flex 0 1 0.
