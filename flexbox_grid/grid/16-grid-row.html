grid-row

            .grid {
                display: grid;
                grid-template-columns: repeat(3, 1fr);
                grid-auto-rows: 50px;
            }
            .grid-row-1 .item-1 {
                grid-row: 3;
            }
            .grid-row-1 .item-3 {
                grid-row: 1;
            }

Esta propriedade altera o posicionamento dos itens para as linhas especificdas.

O item 1 foi deslocado para a linha 3 e o item 3 para a linha 1.

grid-row span

            .grid-row-2 .item-1 {
                grid-row: 1 / 4;
            }
            .grid-row-2 .item-5 {
                grid-row: span 3;
            }
            .grid-row-2 .item-6 {
                grid-row: span 2;
            }

Para o espaçamento funcionar corretamente é essencial determinar o grid-auto-rows. Que neste exemplo está com 50px.

Sem isso, o span fica praticamente imperceptível, porque as novas linhas automáticas não seriam determinadas.

grid-row-start / grid-row-end

            .grid-row-3 .item-2 {
                grid-row: 1 / 3;
            }
            .grid-row-3 .item-4 {
                grid-row-start: 1;
                grid-row-end: 3;
            }
            .grid-row-3 .item-1 {
                grid-row-start: 2;
                grid-row-end: 4;
            }

Como o item 1 foi especificado para da linha 2 até a linha 4, o item 3, que estava na mesma coluna foi descolado para a parte de cima do layout. Sem especificar o posicionamento do item 1, os itens 2 e 4 seriam priorizados. Com isso, o item 1 seria deslocado para a terceira coluna.

[row1]

            .grid-row-4 {
                grid-template-rows: [row1] 1fr [row2] 1fr [row3] 1fr;
            }
            .grid-row-4 .item-2 {
                grid-row: row1 / row3;
            }
            .grid-row-4 .item-4 {
                grid-row: row1 / 4;
            }

Assim como nas colunas, é possível nomear as linhas na propriedade grid-template-rows e usá-los na grid-row.

grid-template-areas

            .grid-row-5 {
                grid-template-areas:
                    "header header header"
                    "sidenav content ads"
                    "sidenav footer footer"
                ;
            }
            .grid-row-5 .item-1 {
                grid-column: span 3;
                grid-row-start: header;
                grid-row-end: header;
            }
            .grid-row-5 .item-2 {
                grid-column: 1;
                grid-row-start: sidenav;
                grid-row-end: sidenav;
            }
            .grid-row-5 .item-3 {
                grid-row: content / content;
            }
            .grid-row-5 .item-4 {
                grid-row-start: ads;
            }
            .grid-row-5 .item-5 {
                grid-row: footer;
                grid-column: span 2;
            }

Neste exemplo não seria necessário especificar o item 5 como grid-row: footer, pois ele já estaria com este posicionamento. Só seria necessário determinar o span 2 para que fossem ocupadas as 2 colunas.
