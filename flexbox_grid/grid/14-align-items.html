align-items: start;

            .grid {
                display: grid;
                grid-template-columns: repeat(3, 1fr);
                grid-template-rows: repeat(2, 100px);
            }
            .item-2, .item-3 {
                grid-row: span 2;
            }
            .item-5 {
                grid-column: span 2;
            }

Foi necessário especificar 100px para as linhas (em grid-template-rows) para criar o espaçamento no container. Sem isso, não teríamos espaço para usar esta propriedade.

A propriedade align-items serve tanto para flexbox, quanto para grid. Lembrando que o correto para o display grid é somente start e no flexobox flex-start.
