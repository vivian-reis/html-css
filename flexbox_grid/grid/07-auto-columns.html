A propriedade grid-auto-columns comecará após as colunas determinadas na grid-template-columns.

Neste exemplo as 2 primeiras colunas tem 1fr cada, a terceira e quinta coluna possuem 50px. Já as colunas quatro e seis estão com 100px.

Foram geradas 6 colunas porque o item-6 foi estabelecido da seguinte maneira:

            .grid-auto-columns-2 > .item-6 {
                grid-column: 6;
            }

grid-auto-columns não definido

            .grid-auto-columns-3 {
                grid-template-columns: 1fr 1fr;
            }
            .item-6 {
                grid-column: 5;
            }

Como o grid-auto-columns não foi definido, as colunas a partir da terceira possuem tamanho correspondente ao tamanho do item.
